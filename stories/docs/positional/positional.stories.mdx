# Positional component

Allows you to position the component anywhere on the screen. ***Depending on what type it is.***

```javascript
import { Positional } from '@avvent/react-layouts';
```

## Props

| Props | Value type | Default | Description |
|:---|:---:|:---:|:---|
| debug | boolean | `false` | Puts a orange background on the parent to visualize where they are. |
| breakpoints | Array<{ [breakpointName]: number or string }> | `default breakpoints` | This is were you can define your own values or just leave the defaults |
| types | **enum[type]** | `relative` | What type position the component should use |
| dimensions | { width: number OR string OR null, height: number OR string OR null } | `{width: null, height: null}` | Columns: How many columns and their size. <br/> Rows: How many rows and their size. <br/> Areas: the areas of the grid. <br/> Auto: What the auto generated rows or columns width or height should be |
| position | { top: number OR string OR null, right: number OR string OR null, bottom: number OR string OR null, left: number OR string OR null } | `{ top: null, right: null, bottom: null, left: null }` | Where the component should be positioned. <br/> relative positions itself accordding to a the next parent that is not a static position item |
| spacing | { margin: number OR string, padding: number OR string } | `{ margin: 0, padding: 0 }` | Margin: The margin of the layout. <br/> Padding: The padding of the layout. |
| z-index | number | 1 | Can be used to shift the component above another |
| children | Can be almost anything | `null` | Go wild |

### Enums
| Enum | Values |
|:---|:---:|
| **type** | `['relative', 'absolute', 'fixed', 'sticky']` |

## Smart props

> It should be noted when a smart props is declared and some of its values are omitted it just reverts to the previously declared value or default value.

| Prop | Syntax |
|:---|:---:|
| dimensions | `dimensions-[breakpointName]` |
| position | `position-[breakpointName]` |
| spacing | `spacing-[breakpointName]` |
| z-index | `z-index-[breakpointName]` |

## Usage example

```jsx
import React from 'React';
import { Positional } from '@avvent/react-layouts';

export default function StickyHeaderExample(props) => {
  return (
    <Positional
      type='sticky',
      position={{
        top: 0
      }}
    >
      <div>
        I act like a sticky navbar
      </div>
    </Positional>
  )
}
