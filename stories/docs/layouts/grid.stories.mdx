# Grid component

The Grid layout component uses CSS Grid as its main layout mechanism.

```javascript
import { Grid } from '@avvent/react-layouts';
```

## Props

| Props | Value type | Default | Description |
|:---|:---:|:---:|:---|
| debug | boolean | `false` | Puts a blue background on the parent and a border around its children to visualize where they are. |
| breakpoints | Array<{ [breakpointName]: number or string }> | `default breakpoints` | This is were you can define your own values or just leave the defaults |
| display | **enum[display]** | `flex` | Whether the layout should be full width to its parent or inline |
| grid | { columns: string, rows: string, areas: [string], auto: { columns: number OR string OR null, rows: number OR string OR null } } | `{columns: '1fr', rows: '1fr', areas: [], auto: { columns: null, rows: null }}` | Columns: How many columns and their size. <br/> Rows: How many rows and their size. <br/> Areas: the areas of the grid. <br/> Auto: What the auto generated rows or columns width or height should be |
| gap | { column: number OR string, row: number OR string } | `{ column: 0, row: 0 }` | What the gap between the grid items should be. Can be just a column gap, row gap or both |
| flow | **enum[flow]** | `row` | In what direction the grid should layout its items. When `dense` is include the grid will try to fill in empty spaces if it can |
| place-items | { justify: **enum[justify1]**, align: **enum[align1]** } | `{ justify: 'stretch', align: 'stretch' }` | Justify: alignment of the items on the main axis. <br/> Align: alignment of the items on the cross axis. |
| place-content | { justify: **enum[justify2]**, align: **enum[align2]** } | `{ justify: 'stretch', align: 'stretch' }` | Justify: alignment of the content on the main axis. <br/> Align: alignment of the content on the cross axis. |
| spacing | { margin: number OR string, padding: number OR string } | `{ margin: 0, padding: 0 }` | Margin: The margin of the layout. <br/> Padding: The padding of the layout. |
| children | `Box` | `null` | The children of this component has to be a Box component |

### Enums
| Enum | Values |
|:---|:---:|
| **display** | `['grid', 'inline-grid']` |
| **flow** | `['row', 'column', 'row dense', 'column dense']` |
| **justify1** | `['start', 'end', 'center', 'stretch']` |
| **align1** | `['start', 'end', 'center', 'stretch']` |
| **justify2** | `['start', 'end', 'center', 'stretch']` |
| **align2** | `['start', 'end', 'center', 'stretch']` |

## Smart props

> It should be noted when a smart props is declared and some of its values are omitted it just reverts to the previously declared value or default value.

| Prop | Syntax |
|:---|:---:|
| grid | `grid-[breakpointName]` |
| gap | `gap-[breakpointName]` |
| place-items | `place-items-[breakpointName]` |
| place-content | `place-content-[breakpointName]` |
| spacing | `spacing-[breakpointName]` |

## Usage example

```jsx
import React from 'React';
import { Grid, Box } from '@avvent/react-layouts';

export default function PageExample(props) => {
  return (
    <Grid
      debug
      gap={{
        column: 20,
        row: 20
      }}
      grid={{
        columns: '1fr 1fr 1fr',
        rows: '100px auto auto',
        areas: [
          'nav nav nav',
          'side content info',
          'footer footer footer'
        ]
      }}

      grid-phone={{
        columns: '1fr',
        rows: '75px auto auto auto',
        areas: [
          'nav',
          'side',
          'content',
          'footer'
        ]
      }}
    >
      <Box area='nav'>
        Navigation
      </Box>

      <Box area='side'>
        I am a sidebar
      </Box>

      <Box area='content'>
        Main content goes here
      </Box>

      <Box area='info' hidden-phone>
        Extras. You won't see me on a phoneðŸ˜’
      </Box>

      <Box area='footer'>
        I am just for SEO.

        OR AM I??
      </Box>
    </Grid>
  )
}

```
