# Flex component

The Flex layout component uses flexbox as its main layout mechanism.

> **NOTE:** The flexbox model relies on its children to make layout decisions.
> Although most of the props are declared on the parent itself and passed it to its children, its important to keep it in mind.

```javascript
import { Flex } from '@avvent/react-layouts';
```

## Props

| Props | Value type | Default | Description |
|:---|:---:|:---:|:---|
| debug | boolean | `false` | Puts a blue background on the parent and a border around its children to visualize where they are. |
| breakpoints | Array<{ [breakpointName]: number or string }> | `default breakpoints` | This is were you can define your own values or just leave the defaults |
| display | **enum[display]** | `flex` | Whether the layout should be full width to its parent or inline |
| useGrid | boolean | `false` | Should the component use a grid-like system |
| columms | number | `12` | If `useGrid` is enabled, how many columns should the grid contain |
| gap | number OR string | `0` | If `useGrid` is enabled, how much spacing should be between items |
| flow | { direction: **enum[direction]**, wrap: boolean } | `{ direction: 'row', wrap: false }` | Direction: The direction of the items flow. <br/> Wrap: should the items go to a new row or column if space is spares |
| place-items | { justify: **enum[justify]**, align: **enum[align1]** } | `{ justify: 'flex-start', align: 'flex-start' }` | Justify: alignment of the items on the main axis. <br/> Align: alignment of the items on the cross axis. |
| place-content | { align: **enum[align2]** } | `{ align: 'flex-start' }` | Align: how all the content should be aligned on the cross axis if there is space available |
| spacing | { margin: number OR string, padding: number OR string } | `{ margin: 0, padding: 0 }` | Margin: The margin of the layout. <br/> Padding: The padding of the layout. |
| children | `Box` | `null` | The children of this component has to be a Box component |

### Enums
| Enum | Values |
|:---|:---:|
| **display** | `['flex', 'inline-flex']` |
| **direction** | `['row', 'row-reverse', 'column', 'column-reverse']` |
| **justify** | `['flex-start', 'flex-end', 'center', 'space-around', 'space-between']` |
| **align1** | `['flex-start', 'flex-end', 'center', 'space-around', 'space-between']` |
| **align2** | `['flex-start', 'flex-end', 'center', 'space-around', 'space-between', 'stretch']` |

## Smart props

> It should be noted when a smart props is declared and some of its values are omitted it just reverts to the previously declared value or default value.

| Prop | Syntax |
|:---|:---:|
| flow | `flow-[breakpointName]` |
| place-items | `place-items-[breakpointName]` |
| place-content | `place-content-[breakpointName]` |
| spacing | `spacing-[breakpointName]` |
